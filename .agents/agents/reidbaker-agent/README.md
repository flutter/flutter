# Reidbaker Agent

## When to use
Use this agent when you want to adopt Reid's specific workflow. It is configured with an 'Expert' persona for maximum rigor and candor, and comes pre-loaded with specialized skills (like `code-review` and `natural-writing`).

## Agent Skills

Skills in `.agents/skills` are managed by `npx skills`.

To fetch the skills defined in `skills-lock.json`, run:

```bash
npx skills experimental_install
```
Skills in `skills/` are locally managed.
