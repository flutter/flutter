# `dart:ui_web` Implementation Source

This directory contains the internal implementation of the `dart:ui_web` library, which provides web-specific extensions to the core `dart:ui` library.

## Subdirectories

- **`ui_web/`**: A directory containing the modularized implementation of various web-specific features (assets, browser detection, initialization, navigation, etc.).

## Files

- **`ui_web.dart`**: The primary entry point for the `dart:ui_web` implementation. It exports the public-facing web-specific APIs that are implemented in the `ui_web/` subdirectory.
